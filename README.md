# 🏥 Hospital Portfolio – AI-Powered Healthcare Platform

A modern **Full-Stack Hospital Management System** designed to simplify healthcare services through an intuitive user interface, secure backend, and intelligent management tools.

The platform enables users to explore hospitals, book appointments, and contact healthcare providers, while providing administrators with a dedicated dashboard to manage hospitals, patients, appointments, and inquiries efficiently.
